# Show Me Scales - Web

**Show Me Scales** is an interactive and visual web application designed to help musicians explore, understand, and memorize scales and chords across different instruments. Built entirely in Vanilla JavaScript, HTML5, and CSS3, it requires no external dependencies.

## Key Features

- **Multi-Instrument Support:**
  - **String Instruments:** Guitar, Bass, Ukulele with dynamic fretboards.
  - **Keyboard:** An interactive 4-octave piano.
  - **Sheet Music:** Classic visualization on a treble clef staff.
- **Customizable Tunings:** Choose from presets (Standard, Drop D, Open G, DADGAD, etc.) or create your own tuning by clicking on the instrument's virtual tuning pegs.
- **3 Modes of Operation:**
  - **Explore Mode:** Browse the dictionary to instantly visualize a scale or chord.
  - **Search by Notes:** Click on the notes of your instrument to find which scale(s) or chord(s) they belong to (reverse search).
  - **Search by Intervals:** Build a formula (e.g., 1, b3, 5) to find the corresponding scale.
- **Harmonization and Dictionary:** Visualize all chords generated by a specific scale (degrees I, II, III...).
- **Advanced UI Tools:**
  - **Split Mode:** Display two instruments side-by-side (handy for comparing a piano and a guitar).
  - **Zen & Fullscreen Mode:** Hide the user interface to focus solely on the instrument.
  - **Canvas Navigation:** Drag to move (Pan) and use the scroll wheel or buttons to zoom in/out.
- **Data Persistence:** Your preferences (key, mode, instruments) are saved locally (`localStorage`).

## Installation and Getting Started

The application is static. However, since it uses the `fetch()` API to load the scale dictionary from an external file (`scales.json`), it is highly recommended to run it via a local web server to avoid browser CORS restrictions.

**Option 1: With Visual Studio Code (Recommended)**

1. Open the folder containing your HTML file in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on your `index.html` file and select **Open with Live Server**.

**Option 2: With Python**

1. Open a terminal in the project folder.
2. Run the command: `python -m http.server 8000` (or `python3`).
3. Open your browser to `http://localhost:8000`.

_Note: If the `scales.json` file is missing, the application is designed to automatically fall back to a default scale database built into the code._

## Usage Recommendations

> **Display:** The application was designed to provide maximum visual information. On mobile devices and tablets, it requires being used in **landscape mode** (a warning screen appears in portrait mode).

- **Change key:** Click on the key button (e.g., "C") in the top right corner.
- **Change Notes/Intervals display:** Use the display button in the top left to toggle between the note name (C, D, E...) or its function in the scale (1, 3, 5...).
- **Tune a string:** Select "Custom" tuning in the instrument menu, then click the note letter at the "nut" (left side of the fretboard) to change it.

## Customization (scales.json)

You can expand the application's scale dictionary by creating or editing a `scales.json` file in the same directory as your `index.html` file.

**Expected format:**

```json
[
  { "name": "Ionian (major)", "intervals": "1, 3, 5, 7, 9, 11, 13", "common": true },
  { "name": "Dorian", "intervals": "1, b3, 5, b7, 9, 11, 13", "common": true },
  { "name": "Whole tone", "intervals": "1, 3, #5, b7, 9, #11", "common": false }
]
```

_The `common` attribute allows filtering the display to hide less common scales from the main view._

## Technical Architecture

- **Rendering:** HTML5 Canvas API (`CanvasRenderingContext2D`). The script dynamically handles resizing and support for high-resolution screens (Device Pixel Ratio).
- **Interactions:** Unified Pointer event listeners (mouse/touch) for drag-and-drop and clicks on the Canvas.
- **Power Management:** Uses the experimental `WakeLock` API to prevent the screen from turning off while using the application.
